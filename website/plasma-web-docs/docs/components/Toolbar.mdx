# Toolbar
Toolbar может отображаться в горизонтальном и вертикальном положении с разными размерами.

## Toolbar

## Использование
Компонент `Toolbar` должен содержать контент передаваемый в `children`.
Для разделения элементов можно использовать ToolbarDivider.

```tsx live
import React from 'react';
import { Toolbar } from '@salutejs/plasma-web';
import { IconEdit } from '@salutejs/plasma-icons';

export function App() {
    return (
        <Toolbar>
            <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            <ToolbarDivider />
            <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
        </Toolbar>
    );
}
```

## Примеры

### Размер Toolbar
Размер Toolbar задается с помощью свойства `size`. Возможные значения свойства: `"l"`, `"m"`, `"s"` или `"xs"`:

```tsx live
import React from 'react';
import { Toolbar } from '@salutejs/plasma-web';
import { IconEdit } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div>
            <Toolbar size="l">
                <Button square size="l" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="l" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="l" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="l" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>

            <Toolbar size="m">
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>

            <Toolbar size="m">
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="s" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="s" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>

            <Toolbar size="xs">
                <Button square size="xs" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="xs" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="xs" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="xs" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>
        </div>
    );
}
```

### Положение Toolbar
Положение Toolbar задается с помощью свойства `placement`. Возможные значения свойства `placement`:
+ `"vertical"` – вертикальное положение;
+ `"horizontal"` – горизонтальное положение;

```tsx live
import React from 'react';
import { Toolbar } from '@salutejs/plasma-web';
import { IconEdit } from '@salutejs/plasma-icons';

export function App() {
    return (
        <div>
            <Toolbar placement="vertical">
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>

            <Toolbar placement="horizontal">
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <ToolbarDivider />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
                <Button square size="m" view="clear" contentType="Left" contentLeft={<IconEdit />} />
            </Toolbar>
        </div>
    );
}
```
